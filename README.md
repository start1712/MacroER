# MacroER

## 一、软件简介

MavroER 是一款功能强大、操作便捷的鼠标键盘宏工具，旨在帮助用户通过自定义脚本、动作录制等方式，模拟鼠标点击、键盘输入等一系列操作，提升操作效率，减少重复操作带来的繁琐。软件支持多种操作模式，适配不同用户的使用习惯，无论是新手还是进阶用户，都能快速上手。

**注：此软件作者为typ3s，start1712为贡献者。**

## 二、核心功能

### 1\. 自定义脚本模拟操作

用户可根据自身需求，自定义编写宏脚本，实现鼠标点击、键盘按键、组合键等各类操作的模拟。脚本可绑定至指定按键，按下绑定按键即可快速触发脚本执行，满足个性化操作需求（如游戏快捷操作、办公重复操作自动化等）。

### 2\. 动作录制功能

若用户不想逐一生成脚本动作，可使用软件的动作录制功能。通过绑定专属按键，控制录制的开始与结束，软件会自动记录录制过程中的所有鼠标、键盘操作，录制完成后可直接保存为脚本，无需手动编写，极大降低操作门槛。

### 3\. 内置代码编辑器

软件内置专业代码编辑器，支持用户直接编写、修改宏脚本。编辑器具备基础语法提示、代码高亮功能，方便进阶用户精准调试脚本，实现更复杂的操作逻辑，满足高阶使用需求。

### 4\. AI 脚本生成（懒人专属）

针对不想手动编写或录制脚本的用户，软件支持 AI 脚本生成功能。用户只需在软件内配置好 AI API 及对应的 API Key，即可通过自然语言描述需求（如“模拟连续点击鼠标左键5次，间隔0\.5秒”），调用 AI 自动生成符合需求的宏脚本，彻底解放双手。

### 5\. 一键去除动作延迟

脚本执行过程中，若存在动作延迟影响操作效率，用户可通过软件内的“一键去延迟”功能，快速清除所有脚本动作的延迟设置，让操作执行更流畅、响应更迅速，适配对操作速度有要求的场景（如游戏、快速办公操作）。

## 三、基础使用流程

1. 启动软件后，进入主界面，可看到“脚本管理”“动作录制”“代码编辑”“AI 配置”四个核心模块；

2. 根据需求选择对应功能：
        

    - 新手推荐：优先使用“动作录制”，绑定录制启停按键，执行需要模拟的操作，录制完成后保存脚本并绑定触发按键；

    - 进阶用户：可通过“代码编辑”手动编写脚本，或通过“AI 配置”填写 API 信息，调用 AI 生成脚本；

3. 脚本生成/录制完成后，可在“脚本管理”中查看、编辑、启用/禁用脚本；

4. 若脚本执行有延迟，点击主界面“一键去延迟”按钮，即可清除所有动作延迟；

5. 按下绑定的触发按键，即可执行对应的宏脚本。

## 四、注意事项

- **软件支持版本：Windows 8.1+ 64位**

- 配置 AI API 时，请确保 API Key 填写正确，且 API 服务处于可用状态，否则无法正常调用 AI 生成脚本；

- 录制动作时，尽量避免无关操作（如误触鼠标/键盘），确保录制内容与需求一致，减少后期编辑成本；

- 脚本执行过程中，建议不要手动操作鼠标/键盘，以免干扰脚本运行，影响操作效果；

- 部分游戏、办公软件可能存在反宏机制，使用前请确认目标软件的使用规范，避免违规。

## 五、常见问题

### Q1：脚本绑定按键后无法触发？

A1：检查按键绑定是否成功，确认绑定的按键未被其他软件占用；同时检查脚本是否处于“启用”状态。

### Q2：录制的动作播放时顺序错乱？

A2：录制过程中若出现误操作，可在脚本编辑界面删除无关动作；若仍有问题，可重新录制动作。

### Q3：AI 无法生成脚本？

A3：检查 API 配置是否正确，API Key 是否有效，网络是否正常；同时确保需求描述清晰、具体，便于 AI 理解。
