核心功能

智能录制

一键录制您的键盘和鼠标操作，自动生成可编辑的脚本。支持录制鼠标移动轨迹、键盘输入和自动添加等待动作。

代码编辑器

内置语法高亮编辑器，支持实时预览和语法提示。让脚本编辑更加直观和高效。

AI 辅助

集成 OpenAI API，通过自然语言描述自动生成脚本。只需告诉 AI 您想做什么，即可获得相应的自动化脚本。

快捷键支持

为每个脚本设置专属快捷键，一键执行自动化任务。支持自定义录制和运行快捷键。

脚本管理

支持多脚本同时管理，快速切换和编辑。支持保存为 JSON 或 Python 格式。

现代界面

采用深色主题设计，界面简洁美观，操作流畅。支持自定义字体大小和界面显示选项。
